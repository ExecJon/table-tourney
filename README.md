Table Tourney

Table Tourney is an iOS app designed to make running small tabletop tournaments simple and organized. It allows players to track match scores, manage brackets, and keep tournaments moving smoothly without needing paper or manual tracking.

The app was built for game nights, local events, and small competitive groups who want a quick way to organize matches and track results.

Table Tourney is available on the App Store.

Features
Quick Tournament Setup

Start a four player 1v1 tournament in just a few taps. Players can be added quickly and the tournament begins immediately.

Match Tracking

Track life totals or victory points with large, easy-to-read counters designed for use around a table.

Tournament Brackets

Table Tourney manages the bracket automatically and records match results as the tournament progresses.

Multiple Matches on One Screen

Two matches can be tracked simultaneously on a single device, making it easy to manage tournament tables or place the device in the center of play.

Automatic Seat Switching

Between rounds, players can switch seats automatically to keep matchups organized and avoid repetition.

Match History

Players can review match results and tournament standings at any time.

Designed for Tabletop Play

Table Tourney works well for a variety of tabletop games including card games, board games, and local tournament events. The interface is designed to be clear and readable from across the table so players can focus on the game instead of managing scores.
